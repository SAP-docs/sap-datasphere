# Importing and Exporting Objects via the Command Line

You can use the SAP Datasphere command line interface, `datasphere`, to import objects to and export objects from your space.

For more information, see [Manage Modeling Objects and Tasks via the Command Line](https://help.sap.com/viewer/d0ecd6f297ac40249072a44df0549c1a/cloud/en-US/6f5c65f209004751aa48f9682ee2ec45.html "Users with a modeler role can use the datasphere command line interface to list, create, update, and delete modeling objects.") :arrow_upper_right:.
